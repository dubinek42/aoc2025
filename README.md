# Advent of Code 2025

## Setup

```bash
poetry install
```

## Running Solutions

Run both parts of a day:
```bash
poetry run aoc <day>
```

Run a specific part:
```bash
poetry run aoc <day> --part 1
poetry run aoc <day> --part 2
```

Run with sample input:
```bash
poetry run aoc <day> --sample
```

Examples:
```bash
poetry run aoc 1              # Run day 1, both parts
poetry run aoc 1 --sample     # Run day 1 with sample input
poetry run aoc 1 --part 2     # Run day 1, part 2 only
```

## Running Tests

Run all tests:
```bash
poetry run pytest
```
